# System Glossary

This glossary explains commonly used terms in WorldCovers.

## Cover
A postal cover (envelope, card, wrapper, or related postal item) that contains one or more postal markings and can be cataloged in the system.

## Marking
Any postmark, cachet, handstamp, meter imprint, or annotation found on a cover. Markings are the core observations used for research and search.

## Cover Marking
The relationship between a specific cover and a specific marking.

## Cover Date
A date extracted from a cover or marking context (for example a cancellation date), stored as a structured date to support filtering and timelines.

## Cover Valuation
Cataloged valuation information associated with a cover, including value amount and supporting notes where applicable.

## Contribution
A proposed catalog change submitted by a user (for example creating or updating a cover, marking, or linked metadata) for moderation.

## Collection
A managed grouping used by administrators to organize and curate catalog records.

## Citation
A reference to a source that supports a claim, date, interpretation, or valuation.

## Reference Work
A bibliographic source (book, article, catalog, archive, etc.) used as authoritative context in documentation and moderation.

## Region / Post Office
Geographic metadata used to identify where a marking or postal event is associated.

## Suggestion Status
The moderation state of a user contribution, typically pending review, accepted, or rejected.
