# Getting Started

Welcome to the WorldCovers Help system.

This page is rendered from markdown files in `frontend/src/help`.

## How to Add Help Content

1. Create a new `.md` file in `frontend/src/help`.
2. Use a top-level heading (`# Title`) for the document title.
3. Save the file and reload the app.

The Help page automatically discovers and renders all markdown files in this directory.
